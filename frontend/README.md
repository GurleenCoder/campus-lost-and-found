# 🎒 Campus Lost & Found Management System

A web-based Lost & Found Management System designed for college campuses. The system helps students report lost items and view items available at the Admin Office, while providing administrators with tools to manage lost and found records efficiently.

## 🚀 Project Status

🛠️ Currently under development.

## 🛠️ Tech Stack

- React
- Vite
- HTML5
- CSS3
- JavaScript (ES6+)

## 📋 Planned Features

- Home Page
- Lost Items
- Found Items
- Report Lost Item
- Item Details
- About Page
- Admin Login
- Admin Dashboard
